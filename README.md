# DodoProject
 基于Dodo开放平台与自己制作的DodoOpenJava制作的一些项目
### 项目列表
Dodo消息与Minecraft消息互通（分为两个，一个支持bukkit系服务端，一个支持bungeecord系）
* 脏话屏蔽加禁言
* 上传资源图片
* 成员加入与退出消息
* 通过表情反应添加身份组
* 有需要的可以自己修改源码构建
